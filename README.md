### 📜 기술 결정 의사결정 (ADR)
- **Zero-Invasive Monitoring**: 소스 코드 수정 없이 바이트코드 조작만으로 성능 측정이 가능한 아키텍처 수립.
- **Annotation-based Filtering**: 에이전트와 비즈니스 로직의 결합도를 낮추기 위해 커스텀 애노테이션 기반 필터링 도입.

---

### 🛡️ 향후 유지보수 가이드 (Maintenance Guide)
- **추적 방지**: 새로운 폴더를 무시하고 싶다면 `.gitignore` 작성 후 반드시 `git rm -r --cached [경로]`를 실행할 것.
- **빌드 관리**: `./gradlew clean`을 습관화하여 `build/` 폴더의 상태를 항상 최신으로 유지할 것.
